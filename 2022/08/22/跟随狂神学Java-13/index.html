内部类

内部类的分类

  1. 成员内部类

  2. 静态内部类

  3. 局部内部类

  4. 匿名内部类

内部类的概念

  • 在一个类的内部再定义一个完整的类

内部类的特点

  • 编译之后会产生独立的字节码文件

    也就是两个类会生成两个class文件

  • 内部类可以直接访问外部类的私有成员，而不破坏封装

  • 可以为外部类提供必要的内部功能组件
